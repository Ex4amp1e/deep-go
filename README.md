# Deep go - homeworks

[![Go](https://github.com/Ex4amp1e/deep-go/actions/workflows/tests.yml/badge.svg?branch=main)](https://github.com/Ex4amp1e/deep-go/actions/workflows/tests.yml)

**deep-go** — репозиторий с решениями домашних заданий по курсу Глубокий Go (Golang).  
Задания организованы в папке [homeworks](./homeworks), по тематике охватывают как базовые, так и продвинутые темы программирования на Go.
